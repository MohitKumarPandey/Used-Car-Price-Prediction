# 🚗 Used Car Price Prediction

This is a beginner Machine Learning project that predicts the price of a used car using **Linear Regression**.

## 📌 Features

- Predict used car price
- Data preprocessing
- Label Encoding
- Linear Regression model
- User input prediction

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 📊 Dataset Features

- Make Year
- Mileage (kmpl)
- Engine CC
- Fuel Type
- Owner Count
- Brand
- Transmission
- Color
- Service History
- Accidents Reported
- Insurance Valid

## 🚀 How to Run

1. Clone this repository.
2. Install the required libraries.

```bash
pip install pandas numpy scikit-learn
```

3. Open the Jupyter Notebook.
4. Run all the cells.
5. Enter the car details to predict the price.

## 📁 Files

- `UsedCarPricePredictionMl.ipynb`
- `dataset.csv`
